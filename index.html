This is the beginning of my Commitment page. Here are some random thoughts about Git.

According to this page, Git is a free and open source distributed version control system designed to handle everything from small to very large projects with speed and efficiency.

These are the steps I take to commit the changes.

  1. git add .
  2. git commit -m "add to index.html"

This is how I create the Repository on Github

  1. Create a Repo on Github
  2. Copy the ssh link name from the green "Code" button in the upper right-hand corner
  3. Go to Terminal on your local desktop and type git clone "ssh link"
  4. Go to Terminal on your server and type git clone "ssh link" in public_html/dev

This is what I do when I am ready to push my commits to GitHub.

  1. git push
This is what I do when I want to install my commits on the server.

  1. git pull
